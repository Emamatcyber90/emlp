What is the response of ML to the following expressions?

(a) 1+2*3
    val it = 7 : int

(b) 5.0-4.2/1.4
    val it = 2.0 : real

(c) 11 div 2 mod 3
    val it = 2 : int

(d) "foo"^"bar"^""

(f) if 6<10 then 6.0 else 10.0
    val it = 6.0 : real

(g) 0XAB+123
    val it = 294 : int

(h) 0xab<123
